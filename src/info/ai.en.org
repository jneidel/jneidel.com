This is a [[https://www.bydamo.la/p/ai-manifesto][/ai page]].
It gives you an honest and detailed overview of how AI and LLMs are used in the
creation of this website.

*My promise:
The writing on this website will always be my own.*
It will never be fully or partially generated by an LLM.

| Editing (writing)       | No                                   |              |
| Article idea generation | No                                   |              |
| Translations            | No                                   |              |
| Cover thumbnails        | Generated by AI, edited by me        | ChatGPT      |
| Images in the article   | No                                   |              |
| Coding                  | Drafted by and refined with an agent | OpenAI Codex |
